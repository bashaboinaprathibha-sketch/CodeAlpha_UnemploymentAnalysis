# CodeAlpha_UnemploymentAnalysis
# Unemployment Analysis with Python

## Project Description
This project analyzes unemployment data in India using Python. It explores unemployment rates across different regions and presents the results using graphs.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Dataset
- Unemployment in India.csv

## Features
- Loads the unemployment dataset
- Displays the first few records
- Checks for missing values
- Shows statistical summary
- Generates Histogram
- Generates Bar Chart
- Generates Pie Chart

## How to Run
1. Install required libraries:
   pip install pandas matplotlib

2. Run:
   python unemployment_analysis.py

## Output
- Histogram of unemployment rates
- Average unemployment rate by region (Bar Chart)
- Top 5 regions by unemployment rate (Pie Chart)
